| 名称 | 全称                | 中文                | 理解                               |
| ---- | ------------------- | ------------------- | ---------------------------------- | --- | --- |
| PWA  | Progressive Web App | 渐进式 web 应用     | 可以按功能逐步引入来编写 web 页面  |
| SSR  |  | 服务端渲染          | 请求 html,服务端渲染完成页面后返回 |
| CSR  |  | 客户端渲染          | 请求 html css js，客户端渲染页面   |
| SSG  |  | 静态生成            | 构建的时候生成所有请求 html        |
